# L-Sky(珞Sky)

L-Sky 是一个基于百度飞浆PaddlePaddle开发的遥感智能解译平台，具有六大特色亮点：

- 多任务解译
- 高精度处理
- 多类型数据
- 自定义空间
- 高友好交互

![image-20220711220725116](README.assets/image-20220711220725116.png)

## 多任务解译

![image-20220711221128121](README.assets/image-20220711221128121.png)

### **4种类型——9种算法**

- 目标提取
    - 道路提取
    - 水体提取
    - 建筑物提取
- 目标检测
    - 操场检测
    - 立交桥检测
    - 飞机检测
- 变化检测
    - 建筑物变化检测
- 地物分类
    - 粗分类
    - 精分类
